  • runMatch(files: Array<string> | Array<{ botkey?: string; file: string; name: string }>, configs?: DeepPartial<Configs>): Promise<any>
    • Defined in https://github.com/StoneT2000/Dimensions/blob/d50f3a0/src/Dimension/index.ts#L299

    Runs a match with the given files and any optional Match.Configs. It rejects if an error occurs. Some errors include MatchDestroyedError which happens when Match.destroy is called.

    This also automatically stores matches into the Database if database is active and configured to save

    Resolves with the results of the match as specified by the Design of this Dimension

    Parameters

    • files: Array<string> | Array<{ botkey?: string; file: string; name: string }>

      List of files or objects to use to generate agents and use for a new match

    • Optional configs: DeepPartial<Configs>

      Configurations that are Design dependent. These configs are passed into Design.initialize Design.update and Design.storeResults

    Returns Promise<any>
