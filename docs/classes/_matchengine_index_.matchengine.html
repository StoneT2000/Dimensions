Class MatchEngine

classdesc

The Match Engine that takes a Design and starts matches by spawning new processes for each Agent.

Functionally runs matches as storing the match causes circular problems (previously Match has Engine, Engine has Match)

    • Defined in https://github.com/StoneT2000/Dimensions/blob/0124128/src/MatchEngine/index.ts#L334

    Run a custom match. A custom match much print to stdout all relevant data to be used by the engine and Dimensions framework. All output after the conclude command from Design.OverrideOptions is outputted is stored as a list of new line delimited strings and returned as the match results.
