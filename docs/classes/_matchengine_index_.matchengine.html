Class MatchEngine

classdesc

The Match Engine that takes a Design and starts matches by spawning new processes for each Agent It returns results while a game is running and returns final results as well. Can start and stop the engine Functionally runs matches as storing the match causes circular problems (previously Match has Engine, Engine has Match)
