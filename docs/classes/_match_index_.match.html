Class Match

classdesc

An match created using a Design and a list of Agents. The match can be started and stopped, and statistics can be retrieved at any point in time. This can be extended if needed

param

The Design used

param

List of agents used to create Match.

param

Configurations that are passed to every run through initialize, update, and storeResults in the given Design
